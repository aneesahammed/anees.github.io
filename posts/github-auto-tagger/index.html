GitHub Auto Tagger

Apr 10, 2019 23:00 · 82 words · 1 minute read GitHub

I keep track of all my work using GitHub project boards.

To simplify, automate & improve GitHub issue management workflow therefore to scratch my own itch, I made simple GitHub Probot app autotagger. It automatically tags GitHub Issues based on the matched pattern.

Simply mention your tags in square brackets […] on Issue title.


Please install autotagger, I’d love to hear your feedback. You can find more details about the Probot from here.
